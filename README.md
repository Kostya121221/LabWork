# Геометрические фигуры

Проект на C++, реализация поиска различных параметров геометрических фигур

## Функции

Проект включает в себя поиск параметров таких фигур как:

+ Прямоугольник:
  + Расчет периметра.
  + Расчет площади.
  + Расчет длины диагонали.


## Структура проекта

+ `main.cpp` - основной файл программы, в котором реализованы все функции.
+ `README.md`: Этот файл.

## Использование 
Скопируйте проект
```
git clone https://github.com/Kostya121221/LabWork.git
```
Скомпилируйте его при помощи вашего компилятора c++(в данном случае g++)
```
g++ main.cpp -o geom
```
Запустите исполняемый файл с готовой программой
```
./geom
```
